This repository **is** the deliverable for the **rtCamp Senior React Engineer**
coding assignment. This file links it back to the job-search application package
for later reference.

## The assignment

- **Company:** rtCamp
- **Role:** Senior React Engineer (100% distributed, async)
- **Brief (source of truth):** https://careers.rtcamp.com/assignments/senior-react-engineer/
- **Deliverable:** "WordCamp Schedule Viewer" — this repo + a hosted Vercel demo.
- **Graded on:** code quality, coding standards, testing practices, and commit
  history. See [`PRD.md`](PRD.md) for the full requirements.

## Submission checklist (do these when the app is done)

1. Confirm the PRD acceptance checklist is fully green (tests ≥60%, demo live, etc.).
2. Deploy to Vercel; copy the production URL.
3. In the job-search repo, paste **this repo's GitHub URL + the Vercel demo URL**
   into `applications/rtcamp/cover-letter.md` (placeholders are already there).
4. Update `tracking/applications.csv` status to `Applied` with the date.
5. Submit via the rtCamp assignment/careers channel with the CV `.docx`.
